Privacy in Public Spaces Based on Augmented Reality

The project is about exploring the privacy of one in public spaces. Geometrical components are studied to generate a cluster to form a shell, ensuring privacy eventually. The field logic is then applied using the generated geometrical clusters to develop the personal boundary in the virtual space. Unity is used along with the geometrical components to create the desired boundary, using the user as an attractor. To visualise the formulated virtual boundary augmented reality, AR is used and deployed in the HoloLens. Finally, the project is tested to determine the privacy level using a questionnaire with a group of users.
